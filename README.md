# AI-Based YouTube Video Summarization System
AI-powered system that summarizes YouTube videos into multilingual text and audio using Whisper, LLaMA-3, and Streamlit.
## Project Structure

│
├── app.py
├── requirements.txt
├── README.md
└── project-demo.png
## Features

• Automatic YouTube video transcription  
• AI-based video summarization  
• Multilingual translation support  
• Text-to-Speech audio summary  
• Simple Streamlit interface  

## How It Works

YouTube URL
   ↓
Audio Extraction (yt-dlp)
   ↓
Speech-to-Text (Whisper)
   ↓
LLM Summarization (LLaMA-3 via Groq)
   ↓
Translation
   ↓
Text-to-Speech (gTTS)
   ↓
Streamlit Web Interface
